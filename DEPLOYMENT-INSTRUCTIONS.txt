DEPLOYMENT STEPS:
-----------------

1. UPLOAD TO CPANEL
   - Log in to your cPanel account
   - Navigate to File Manager
   - Upload deeskinstore-deployment-final.tar.gz to your home directory
   - Extract the archive

2. INSTALL DEPENDENCIES
   - Open Terminal in cPanel
   - Navigate to the project directory
   - Run: npm install

3. CONFIGURE ENVIRONMENT
   - Copy .env.example to .env
   - Update with your actual values:
     * VITE_SUPABASE_URL (already configured)
     * VITE_SUPABASE_ANON_KEY (already configured)
     * VITE_PAYSTACK_PUBLIC_KEY (add your Paystack key)

4. BUILD FOR PRODUCTION
   - In Terminal, run: npm run build
   - This creates the dist/ folder with production files

5. DEPLOY TO PUBLIC_HTML
   - Update .cpanel.yml with your actual username
   - The deployment script will copy dist/* to public_html
   - Or manually copy dist/ contents to public_html

6. VERIFY DEPLOYMENT
   - Visit your website
   - Check admin login at: yourdomain.com/admin
   - Admin credentials: From your database setup
